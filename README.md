# Task Manager - Spring Boot

A simple task management web application built using Spring Boot.

## Features
- Create tasks
- Search tasks
- Delete tasks
- REST API backend
- Simple UI using HTML + JavaScript

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven

## How to Run

1. Clone the repository
2. Open in IntelliJ
3. Run TaskmanagerApplication
4. Open browser

http://localhost:8080
